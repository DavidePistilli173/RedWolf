Detailed Description

Base class for all RedWolf applications. To be extended by users of the library.

Member Function Documentation

◆ run()

void rw::core::BaseApplication::run ( )

Run the application.

The main loop does not use a rw::time::Timer because it must be executed by the main thread to support input events in graphical applications.
